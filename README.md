📝 QT-Word: MiniOffice Suite
Una aplicación de escritorio ligera inspirada en herramientas de ofimática, desarrollada con Python y el framework PyQt6. Este proyecto se enfoca en la creación de interfaces de usuario intuitivas y la implementación de componentes reutilizables.

🚀 Características Principales
Editor de Texto: Interfaz funcional para la gestión de documentos.

Contador de Palabras: Widget personalizado integrado para el análisis de texto en tiempo real.

Reconocimiento de Voz: Implementación de dictado por voz para mejorar la accesibilidad y productividad.

Instalador Integrado: Generación de ejecutable mediante el script de configuración .iss incluido.

🛠️ Tecnologías Utilizadas
Lenguaje: Python 3.x

Interfaz Gráfica: PyQt6 (Qt for Python)

Gestión de Dependencias: Pipenv (Pipfile)

Instalación: Inno Setup (para el archivo .iss)

📦 Instalación y Uso
Clonar el repositorio:

Bash
git clone https://github.com/Alejandro-Shadow/QT-Word.git
Instalar dependencias:

Bash
pipenv install
Ejecutar la aplicación:

Bash
python QT6PracticaFinal.py
